#subsd

# Decriptions
Script to check all sub folders of /sdcard

# Tools made this

- mksh
- Terminal Emulator
- Material files

# Tested

a. Platform

- Termux
- Term OnePlus

b. Device

- j2ltedd (Samsung J2 / SM-J200G)
- fog (REDMI 10C)

c. Target

- All android devices

# How to resolve known issue at ztmexluis

a. HISTSIZE BAD NUMBER
type ``` unset HISTSIZE```

b. REQUIRED FILE NOT FOUND
type ``` termux-fix-shebang subsd.sh```
